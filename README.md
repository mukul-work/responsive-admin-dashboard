# Gravix – Responsive Admin Dashboard

A fully responsive admin dashboard UI built using **HTML5 and CSS3**, focused on layout architecture, clean component structure, and modern responsive design principles.

This project demonstrates strong fundamentals in CSS Grid, Flexbox, and structured UI development without using any frameworks.

[Live URL](https://mukul-work.github.io/responsive-admin-dashboard/)

---

## Features

- Fully responsive layout (Desktop → Tablet → Mobile)
- CSS Grid-based page structure
- Sidebar navigation
- Header with search and user profile
- Project cards with action icons
- Announcements section
- Trending users panel
- Clean CSS reset implementation
- Organized asset structure

---

## Layout Architecture

### Page Structure

- `body` → CSS Grid (Sidebar + Main Container)
- `main` → CSS Grid (Left Panel + Right Panel)

### Responsive Breakpoints

- `1100px` → Sidebar width adjustment
- `950px` → Main layout stacks vertically
- `700px` → Sidebar converts to horizontal layout
